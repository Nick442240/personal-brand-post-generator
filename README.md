# Генератор постов для личного бренда

Локальное мини-приложение на Flask, которое помогает собрать идею поста для соцсетей: заголовок, вступление, структуру, призыв к действию и хэштеги.

## Функция сайта

Сайт принимает данные пользователя и собирает шаблонный пост для личного бренда. Результат зависит от выбранной площадки и стиля текста: экспертного, дружелюбного, продающего или вдохновляющего.

Все работает локально: без API, баз данных, внешних сервисов и реальных персональных данных.

## Редактор и проверка поста

На странице результата есть дополнительная функция: редактор поста перед публикацией. Пользователь может вручную изменить сгенерированный текст, сохранить правки, увидеть количество символов и проверить пост по простому чек-листу качества.

Кнопка «Скопировать пост» копирует финальную отредактированную версию текста.

## Как работает редактор поста

- Пользователь получает сгенерированный пост и редактирует его в текстовом поле.
- Сайт сразу обновляет счетчик символов и показывает подсказку по выбранной площадке.
- Чек-лист проверяет наличие заголовка, вступления, CTA, хэштегов и подходящую длину текста.
- После нажатия «Сохранить изменения» текущий текст считается финальной версией поста.

## Вход → Действие → Результат

- Вход: пользователь вводит тему поста, целевую аудиторию, площадку и стиль текста.
- Действие: Flask-приложение проверяет поля и собирает текст по локальному шаблону.
- Результат: пользователь видит заголовок, вступление, структуру, CTA, хэштеги и единый блок текста для копирования.

## Запуск

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Запустите приложение:

```bash
python app.py
```

3. Откройте сайт в браузере:

```text
http://127.0.0.1:5000
```

## Примеры тестовых данных

Пример 1:

- Тема поста: как эксперту начать вести Telegram
- Целевая аудитория: начинающие консультанты
- Площадка: Telegram
- Стиль текста: Дружелюбный

Пример 2:

- Тема поста: как упаковать консультацию
- Целевая аудитория: психологи и коучи
- Площадка: VK
- Стиль текста: Продающий

Пример 3:

- Тема поста: почему личный бренд помогает находить клиентов
- Целевая аудитория: фрилансеры
- Площадка: Блог
- Стиль текста: Экспертный

## Пример использования редактора

1. Введите тестовые данные на главной странице и нажмите «Сгенерировать идею».
2. На странице результата измените текст в поле «Единый текст для копирования».
3. Посмотрите счетчик символов и статусы чек-листа.
4. Нажмите «Сохранить изменения».
5. Нажмите «Скопировать пост», чтобы скопировать финальную версию.

Проект работает полностью локально и не использует внешние API, Telegram, ВКонтакте, базы данных или другие внешние сервисы.
