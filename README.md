# Генератор постов для личного бренда

Локальное мини-приложение на Flask, которое помогает собрать идею поста для соцсетей: заголовок, вступление, структуру, призыв к действию и хэштеги.

## Запуск

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Запустите приложение:

```bash
python app.py
```

3. Откройте сайт в браузере:

```text
http://127.0.0.1:5000
```

Все данные обрабатываются локально. Внешние API и нейросети не используются.
